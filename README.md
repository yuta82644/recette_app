
## アプリ概要
- アプリ名: Recette

## 開発言語
- Ruby 3.0.1
- Ruby on Rails 6.1.6

## 就職Termの技術
- devise
- Ajaxコメント機能
- AWS

## カリキュラム外の技術
- ransack
- カレンダー機能

# 実行手順

```
$ https://github.com/yuta82644/recette_app.git
$ cd recette_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s

```
